做一个练习小提琴识谱, 识音的网站

全局配置

1. 选择小提琴把位(可以选择多个)
2. 选择琴弦(可以选择多个)
3. 小提琴的声音可以由用户自己上传(需要后端, 所以暂时不做)

功能A: 出题模式:

小提琴声音, 音名, 五线谱上的音符, 小提琴把位 + 指位 + 弦名
前3者互为一对一关系,
前3者和 小提琴把位 + 指位为 一对多关系

题目是其中任意一个, 让用户选择正确

配置 音名 + (小提琴把位 + 指位 + 弦名) 时

题目可能是下面的
```
音名: A4
则选项有
A. (1把位 A弦 0指)
B. (1把位 E弦 0指)
C. (1把位 D弦 0指)
D. (1把位 G弦 0指)
```

功能B: 练习模式

鼠标点击某个音名, 播放小提琴对应的声音 + 显示五线谱 + 小提琴把位 + 指位 + 弦名
